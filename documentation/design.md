## Wireframe Notes

### Homepage
- **Hero Section:** Large, impactful image with clear value proposition
- **Quick Stats:** Immediate credibility with key metrics
- **Benefits Section:** 6 key benefits in a grid layout (3x2 on desktop, stacked on mobile)
- **Testimonials:** Carousel with driver photos and quotes
- **CTA Section:** Prominent call-to-action before footer

### Apply Page
- **Multi-Step Form:** Clear progress indicator showing current step
- **Sidebar:** Helpful information and requirements checklist
- **Form Layout:** Two-column layout for related fields (e.g., First/Last Name)
- **Validation:** Real-time validation with clear error messages
- **Mobile:** Single column layout, sidebar moves below form

### Pay Rates Page
- **Interactive Calculator:** Top section for user engagement
- **Tabbed Interface:** Different pay structures organized in tabs
- **Comparison Table:** Clear comparison with industry standards
- **Visual Elements:** Charts and graphs for data visualization
- **Multiple CTAs:** Strategic placement throughout the page

### Responsive Considerations
- All wireframes designed mobile-first
- Desktop: Multi-column layouts, sidebars, larger spacing
- Tablet: Adjusted column counts, maintained hierarchy
- Mobile: Single column, stacked elements, touch-friendly targets (min 44px)
