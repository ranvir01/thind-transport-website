# üìã Quick Google Forms Setup Steps

1. Go to **forms.google.com**
2. Click **+ Blank** to create new form
3. Set the title: **"Thind Transport Website - Information Gathering"**
4. Add description from the top of this document
5. Create sections using the **"Add section"** button (Tt icon)
6. For each question:
   - Copy the question text
   - Select the field type (Short answer, Paragraph, Multiple choice, Checkboxes)
   - Add options if applicable
   - Set Required if marked
   - Add description/placeholder text if provided
7. Enable **"File upload"** for Q102 (requires respondent to sign in)
8. In Settings:
   - Collect email addresses: **ON** (optional)
   - Limit to 1 response: **ON**
   - Allow response editing: **ON**
9. Click **Send** and share the link
