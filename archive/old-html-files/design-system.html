Dark Mode Support

Toggle dark mode with the button in the top right corner

Automatic Adaptation

All components automatically adapt to dark mode using CSS variables. The design system includes carefully selected OKLCH colors that maintain contrast ratios in both modes.

  • Perceptually uniform colors
  • Better accessibility
  • Consistent experience

Implementation

/* Dark mode styles are automatically applied */ .dark { --background: oklch(0.141 0.005 285.823); --foreground: oklch(0.985 0 0); --card: oklch(0.21 0.006 285.885); /* ... more variables ... */ } /* Toggle dark mode */ document.documentElement.classList.toggle('dark');
